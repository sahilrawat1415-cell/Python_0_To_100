# 🐍 100 Python Programs – Beginner to Advanced

This repository contains solutions to **100 Python programming questions** designed to build strong problem-solving skills from scratch.

The problems start with basic input/output and gradually progress to loops, numbers, strings, arrays, and challenging programming concepts.

## 📌 Repository Goal

- Learn Python fundamentals
- Improve logical thinking
- Practice problem-solving
- Build consistency through coding
- Prepare for Data Structures & Algorithms (DSA)
